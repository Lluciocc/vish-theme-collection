# vish-theme-collection
A centralized repository for managing and storing Vish themes.

## Vish repository

This repository only stores themes made by the community.
You can find the main Vish repository here:https://github.com/Lluciocc/vish

## Vish on Flathub

You can install Vish via Flathub: https://flathub.org/en/apps/io.github.lluciocc.Vish

## Share Your theme
If you want to contribute to Vish itself, you're in the wrong repository.
Please go here instead:
https://github.com/Lluciocc/Vish/wiki/Contributing

If you want to share your theme:

1. Open a Pull Request with your .yml file and the README.md edited with your changes.
2. Make sure your colors are well balanced and readable
3. The name field must match the filename
4. Add a short description and (optionally) a screenshot
